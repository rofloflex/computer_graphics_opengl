# Computer Graphics (OpenGL, C++)

Учебный репозиторий по лабораторным работам по компьютерной графике.

## Структура репозитория

- `main` — стабильная версия.
- `lab-01`, `lab-02`, ... — отдельные ветки для каждой лабораторной.

## Как работать с лабораторными

1. Создать ветку от `main`:
   git checkout main
   git pull
   git checkout -b lab-01

2. После сдачи лабораторной — слить в `main`:
    git checkout main
    git merge lab-01
    git push
